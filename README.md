# ocrsamples
OCR SDK Sample (Google ML kit, Card.IO, Tesseract)

CameraX를 이용하여 세 개의 OCR SDK의 샘플을 구현.
가장 기본적인 내용만을 구현하여 세 개의 SDK 모두 하나의 액티비티 클래스로 이루어져 있다.
